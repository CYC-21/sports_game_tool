# 色票與顏色定義（可編修）

> 修改後請同步更新 `web/css/app.css` 的 `:root`，或將本檔交給協作者套用。  
> **最後同步**：2026-03-21 — 與 `web/css/app.css` 一致（主卡黃 `--card-yellow: #f7b728`；第二節已改為 CSS 變數名）。

---

## 一、`:root` 主色票

| 變數名稱 | 值 | 用途摘要 |
|----------|-----|----------|
| `--page-bg` | `#ffffff` | 頁面主底色 |
| `--page-elevated` | `#f3f5f9` | 篩選列、模式列、列表區、表單欄底等略抬高的區塊 |
| `--chrome-line` | `rgba(7, 37, 163, 0.14)` | 邊框／分隔線 |
| `--ink-navy` | `#0725a3` | 深藍主色（字、左條、主按鈕底等） |
| `--label-blue` | `#1033ff` | 「場館」標籤字色 |
| `--card-yellow` | `#f7b728` | 一般賽事卡底色、模式「選中」底 |
| `--card-cream` | `#fff1ab` | 未確認／tentative 賽事卡底 |
| `--card-gray` | `#d3d3d3` | 延期賽事卡底 |
| `--card-ink` | `#0725a3` | 賽事卡內主文字 |
| `--card-ink-muted` | `rgba(7, 37, 163, 0.72)` | 賽事卡內次要字、主客標籤等 |
| `--text` | `#141826` | 全站主內文色 |
| `--text-muted` | `#5a6272` | 次要說明、輔助文字 |
| `--accent` | `#1a4fd4` | 連結、次要按鈕、強調互動 |
| `--green` | `#0d7a45` | 狀態點「已確認」、成功傾向 |
| `--red` | `#b42318` | 延期標籤、錯誤、必填星號（`web/js/app.js` 的 `*`） |
